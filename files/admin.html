📧 Order Confirmation Emails

Automatically email customers when they place an order using EmailJS (free — 200 emails/month).

Setup:
1. Go to emailjs.com → Sign up free
2. Add an Email Service (connect Gmail or any email)
3. Create an Email Template — use variables: {{to_email}}, {{customer_name}}, {{order_id}}, {{order_total}}, {{order_items}}, {{payment_method}}, {{delivery_address}}
4. Copy your keys from Account → API Keys

🛡️ Firestore Security Rules

Copy these rules into Firebase Console → Firestore → Rules and click Publish. These rules lock down your database so only signed-in admins can modify data.

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // Products: anyone can read, only admin can write
    match /products/{doc} {
      allow read: if true;
      allow write: if request.auth != null;
    }

    // Orders: anyone can create, only admin can read/update/delete
    match /orders/{doc} {
      allow create: if true;
      allow read, update, delete: if request.auth != null;
    }

    // Settings: anyone can read, only admin can write
    match /settings/{doc} {
      allow read: if true;
      allow write: if request.auth != null;
    }

    // Internal ping
    match /_ping/{doc} {
      allow read, write: if request.auth != null;
    }
  }
}
🔥 Firebase Database

Connect your store to Firebase Firestore so products, orders, and stock sync across all devices in real-time — no more data lost between browsers.

How to get these values:
1. Go to console.firebase.google.com
2. Create a project → Add a Web App → copy the config values below
3. In Firestore Database → Rules, paste the secure rules from the card below
4. In Firebase Console → Authentication → Get Started → Email/Password → Enable
5. In Authentication → Users → Add user — add your email & password
